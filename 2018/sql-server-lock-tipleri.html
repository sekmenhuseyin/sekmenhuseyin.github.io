SQL Server Lock Tipleri

Shared Locks

Okuma işlemi sırasında oluşturulan lock biçimidir.

Exclusive Locks

Ekleme, güncelleme veya silme işlemler sırasında oluşturulur. Başka bir oturum tarafından değiştirilmesini engeller. İşlem bitinceye kadar aktif olurlar

Update locks

Exclusive lock öncesi çalışan locktur. Güncelleme yada silme işlemi için gerekli satırları ararken aktif hale gelir. Doğru satırı bulursa lock biçimini exclusive olarak değiştirir

Intent locks

Exclusive lock yapılan bir nesneni kullanan diğer nesnelerde aktif olan lock biçimidir.
