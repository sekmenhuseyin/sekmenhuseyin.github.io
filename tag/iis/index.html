Server 2008 IIS7’de ayrıntılı hata bilgisi göstermek – 500 Internal Server Error

Server 2008’de yüklü gelen IIS 7,5 güzel özellikleri olmasına rağmen kullanımını öğreninceye kadar belli bir süre geçiyor. Ben en çok hata ayıklarken bu yeni IIS’den usanmıştım. Web sitesini seçince gelen simgelerden ASP simgesine çift tıklayınca gelen ekrandan “Send Errors To Browser” ayarını true yapınca hata gözükecek sanıyordum. Meğer bir değişiklik daha yapmak gerekiyormuş.
